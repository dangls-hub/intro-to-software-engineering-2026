# Hệ thống Quản lý chung cư BlueMoon (AMS)

## Giới thiệu dự án
**BlueMoon AMS** là hệ thống phần mềm tập trung nhằm chuyển đổi số quy trình quản lý thông tin hộ dân và quản lý thu phí từ mô hình vận hành thủ công. Hệ thống được xây dựng mới hoàn toàn nhằm tối ưu hóa hiệu suất nghiệp vụ tại chung cư Blue Moon.

##  Mục tiêu dự án
* **Số hóa nghiệp vụ**: Thiết lập hệ thống lưu trữ tập trung cho dữ liệu hộ dân và lịch sử tài chính.
* **Đảm bảo tính sẵn sàng**: Hệ thống đạt chỉ số sẵn sàng phục vụ 24/7.
* **An ninh thông tin**: Triển khai cơ chế phân quyền và bảo mật dữ liệu dân cư mức độ cao.

## Tính năng chính
* **Quản lý dân cư**: Xây dựng và quản lý cơ sở dữ liệu chi tiết về các hộ dân tại Blue Moon.
* **Quản lý tài chính**: Tự động hóa tính toán và theo dõi các danh mục thu phí.
* **Tích hợp thanh toán**: Hỗ trợ giao dịch trực tuyến thông qua Dynamic QR Code và tích hợp API Payment Gateway.
* **Phạm vi**: Phần mềm nội bộ tập trung vào nghiệp vụ quản lý thu phí quy mô nhỏ.

## 🛠 Công nghệ sử dụng
* **Ngôn ngữ & Framework**: Java (Spring Boot / JavaFX).
* **Hệ quản trị CSDL**: MySQL, đảm bảo tính toàn vẹn cho dữ liệu tài chính.
* **Quản lý mã nguồn**: Sử dụng GitHub để quản lý source code và phiên bản.
